About

This app is the deployment of a Python programme that takes an address in Flanders region, Belgium and renders a 3D plot of the corresponding house. Because of space limitations this app is restricted to addresses in the city centre of Ghent and postal code 9000.

How does it work?

  1. Retrieve Lambert 72-coordinates of address from API call
  2. Select corresponding Canopy Height Model
  3. Retrieve house polygon of address from API call
  4. Create a mask of the polygon from the Canopy Height Map
  5. Estimate dimensions from house polygon
  6. Plot a surface map of the house mask

Have fun!
